# Sahayak — A Digital Help Desk.

Sahayak is a voice-first digital help desk built during Hack For Tomorrow 2026, designed to simplify access to essential public services such as healthcare, government welfare, education, employment, legal aid, and emergency support.

The platform focuses on accessibility, simplicity, and multilingual interaction to help users easily discover and access the services they need.


## Features

* Voice-first interaction flow
* Personalized public service assistance
* Multilingual support
* Step-by-step guidance for services
* Emergency SOS assistance
* Accessibility-focused user experience
* Responsive mobile-friendly interface
* Real-time service support and navigation
